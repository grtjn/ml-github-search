Search through MarkLogic repositories on Github

About

This website allows you to find MarkLogic-related repositories on Github more easily. It is built using slush-marklogic-node with all its out of the box features, and the 'blue' theme. It uses ml-github-api to access the Github API to find MarkLogic-related repositories, and gather information for the benefit of searching inside MarkLogic. It uses the schedule provided in that project to refresh the contents inside this MarkLogic database every hour. It leverages ml-highcharts-ng to draw below charts, and d3-cloud-ng to draw the keywords tag cloud. Dashboard and Seach page use the same search context, so drilling down in one, will change both.
